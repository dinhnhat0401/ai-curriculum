## The Shift

Phases 1-3 were about learning. Phase 4 is about shipping.

The difference:
- **Learning**: "I built a RAG system" (great)
- **Shipping**: "I built a RAG system that saves Company X 20 hours/week on document review" (money)

Everything in Phase 4 is oriented around one question: **does this solve a real problem for someone who will pay for it?**
