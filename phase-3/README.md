# Phase 3: Business Stack

**Weeks 9-12 | ~32 hours | You can build AI systems. Now learn to ship them.**

---

```
    Phase 2: "It works on my laptop"
    Phase 3: "It works in production, at scale, with monitoring, under budget"
```

## Modules

| Module | What You Build | Why It Matters |
|--------|---------------|----------------|
| **AWS Bedrock** | Enterprise RAG on managed AWS services | Enterprise clients run on AWS. Bedrock = no infra to manage. |
| **Production** | Dockerized, monitored FastAPI service with CI/CD | The difference between a prototype and a product. |
| **Evaluation** | Production eval suite + cost optimization | Continuous quality monitoring + keeping costs under control. |

## After This Phase

You can:
- Deploy AI on enterprise infrastructure (AWS)
- Ship a containerized, monitored, CI/CD-deployed service
- Measure and optimize cost per query
- Tell a client exactly what it costs and how well it works
